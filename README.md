## Enigma Cipher Machine

Горбачева Маргарита Валерьевна | студентка группы БПИ-245 ФКН НИУ ВШЭ

#### Описание проекта
Данный курсовой проект представляет собой модель шифровальной машины «Энигма» — электромеханического устройства, использовавшегося для шифрования и дешифрования секретных сообщений в первой половине XX века.

#### Цели проекта

* Изучение принципов работы шифровальной машины «Энигма».
* Разработка модели, имитирующей работу оригинальной машины.
* Исследование криптографической устойчивости шифра «Энигма».

#### Технологии

* Язык программирования -- C++.
* Библиотеки для работы с текстовыми данными.
* Возможно использование фреймворков для визуализации работы машины.

#### Структура проекта

1. `Документация` — описание принципов работы «Энигмы», ключевые компоненты и их функции.
2. `Моделирование` — реализация модели шифровальной машины.
3. `Тестирование` — проверка работы модели на различных текстах и шифрах.
4. `Примеры использования` — демонстрация работы модели на конкретных примерах.
